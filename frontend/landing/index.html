    Run it on your own server. Your data stays yours.

Setup in 5 Minutes

Three APIs.
One .env file.
Done.

  • ✓ Data source — connect any keyword or analytics API
  • ✓ AI model — use OpenAI, Gemini, Claude, or any provider
  • ✓ Scraper — optional self-hosted data extraction
  • ✓ Deployment — run locally or deploy anywhere
# Connect your data source API Use any provider or your own backend DATA_SOURCE_USERNAME=your_username DATA_SOURCE_PASSWORD=your_password
# Choose your AI provider AI_PROVIDER=openai # openai | gemini | anthropic AI_MODEL=gpt-4o OPENAI_API_KEY=sk-... # Or for Gemini: GEMINI_API_KEY=AIza... # Or for Claude: ANTHROPIC_API_KEY=sk-ant-...
# run your own scraping engine # github.com/MpMeetPatel/sniffhunt-scraper SNIFFHUNT_URL=http://localhost:8080 SNIFFHUNT_GEMINI_KEY=AIza...
